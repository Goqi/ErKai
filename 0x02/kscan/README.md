# [kscan](https://github.com/lcvvvv/kscan)代码分析

本文将从代码层面深入分析kscan项目。kscan是lcvvvv开发的一款全方位扫描器，具备端口扫描、协议检测、指纹识别，暴力破解等功能。支持协议1200+，协议指纹10000+，应用指纹20000+，暴力破解协议10余种。

本文创建于2022年11月11日，最近的一次更新时间为2022年11月11日。
