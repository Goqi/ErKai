# [scaninfo](https://github.com/redtoolskobe/scaninfo)代码分析

本文将从代码层面深入分析scaninfo项目。scaninfo是redtoolskobe开发的一款开源、轻量、快速、跨平台的红队内外网打点扫描器。包含更好的web探测、指纹识别和更好的报告输出。
